# RFM-анализ клиентов (Olist E-commerce)

RFM-сегментация клиентов на основе SQL (CTE, оконные функции NTILE) и Python (pandas, matplotlib).

## Стек
- SQL (MS SQL / SSMS): CTE, оконные функции (NTILE), DATEDIFF
- Python: pandas, matplotlib

## Что сделано
- Рассчитаны метрики Recency, Frequency, Monetary для 99 440 клиентов
- Клиенты разбиты на квартили по каждой метрике (NTILE(4))
- Проанализированы размеры и характеристики 16 RFM-сегментов
- Построена визуализация среднего чека по сегментам

## Результат
Сегмент "Champions" (111) показывает средний чек ~390 BRL против ~43 BRL у "Lost" (444) — разница почти в 9 раз, что подтверждает значимость сегментации для таргетированных маркетинговых кампаний.
